# RailMaint L5 HVAC

Sistema de registro de manutenção HVAC para trens da Linha 5 Lilás.

Aplicação desenvolvida para registrar manutenções:

- Preventivas
- Preditivas
- Corretivas

em sistemas de ar condicionado de trens metropolitanos.

Funcionalidades:

• Registro de manutenção
• Histórico de intervenções
• Dashboard de falhas
• Geração de relatório técnico
• Instalação como aplicativo Android (PWA)

Tecnologias utilizadas:

HTML
CSS
JavaScript
LocalStorage
PWA (Progressive Web App)
